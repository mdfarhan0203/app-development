# Database?
-  database is a collection of data that is stored in a computer system.
- database is used to store data in a way that is easily accessible and can be used to retrieve data.

# Types of database
- relational database
- non-relational database

# Relational database
- relational database is a database that uses tables to store data.
- relational database is a database that uses tables to store data.
- Example - MySQL, PostgreSQL, Oracle

# Non-relational database
- non-relational database is a database that does not use tables to store data.
- Example - MongoDB, CouchDB, Cassandra

# Database management system
- database management system is a software that is used to manage a database.
- Example - MySQL, PostgreSQL, Oracle
